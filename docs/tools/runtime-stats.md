# runtime_stats

**Short Description:** Get runtime statistics

**Full Description:** Get runtime statistics including memory usage, thread count, and GC info. Use to monitor system health and diagnose memory issues.

**Parameters:** None

**Returns:** Runtime statistics including memory, threads, and GC info

**Example Usage:**
```lisp
(runtime_stats)
```

**Notes:** Useful for monitoring system health and diagnosing memory or performance issues.
