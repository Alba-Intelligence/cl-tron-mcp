# SBCL Debugging MCP Repository Guidelines

This document provides guidelines for AI agents working on the SBCL Debugging MCP project - a Model Context Protocol server that enables deep debugging, introspection, profiling, and hot code reloading for SBCL Common Lisp applications.

## Modular Documentation

This guide has been split into focused modules for better token efficiency:

- **[Getting Started](getting-started.md)** - Quick start, discovery, and first steps
- **[Workflows](workflows.md)** - Common workflows and patterns
- **[Tool Reference](tool-reference.md)** - Complete tool catalog by category
- **[Conventions](conventions.md)** - Coding style, naming, and best practices
- **[Transport & Logging](transport-logging.md)** - Stdio, HTTP, WebSocket, and logging requirements
- **[Troubleshooting](troubleshooting.md)** - Common issues and solutions
- **[Token Optimization](token-optimization.md)** - Principles for reducing token usage
